3D Face Reconstruction Advanced

I. INTRODUCTION

This project is aiming to reconstruct human face from a single facial image. Further, I hope to find a way to use these reconstructed faces, I decide to import the reconstructed faces into unity. The whole project is divided into four sub-tasks: data processing, face model, network and unity processing.

A. Data processing

In this part, we want to get images of human faces as dataset. Firstly, we use CACD 2000 dataset as our image source, dlib is used to detect face and remove those images which have 0 or more than 1 human face. Secondly, we use Opencv to resize and normalize original facial images. Thirdly, face-alignment is applied to recognize the face on resized images to get 68 facial landmarks. The whole dataset’s item consists of 224224 facial image and 682 facial landmarks.

B. Face model

In this part, we decide to use Basel Face Model as our face reconstruction tool. In order to complete the face reconstruction. Firstly, we need 230 coefficients and split them into shape, expression, texture, rotation, translation and scale params, use these params to reconstruct human face. Secondly, generate human face mesh through Basel Face Model from face shape, expression and texture params. Thirdly, we need to add lighting to the face mesh surface. Fourthly, we will scale, rotate and translate the face mesh to adjust its position and scale.

C. Network and loss function

We build and train a residual neural network to generate coefficients from a facial image that can be used to generate 3D face through Basel Face Model. Firstly, we built a ResNet-34 as our network. Secondly, in the training process, we project 3D face object to 2D plane to calculate the difference between reconstructed faces and target faces. Thirdly, per-pixel color and landmark differences are used to compute loss and train network.

D. Unity processing

Finally, to utilize the network. We need to run the 3D face reconstruction process in unity. I use Diagnostics class in C sharp to achieve this purpose, this class provides access to local and remote process and enable us to start system processes.

II. DATA PROCESSING

In this project, I used CACD 2000 dataset as training data source.

Firstly, we need to resize these images into 224*224 size and normalize the resized images.

Secondly, in the training process, we need face landmarks to locate the contour positions of face, which means we need to recognize and detect the 68 facial feature points in a face image. To get the 68 face landmarks, we use face-alignment to detect 2D facial landmarks in facial image.

The 68 facial landmarks have two dimensions represents the x and y position in the 224*224 face image.

IV. Network and Loss Function

In this part, we build our residual network. Our network can transfer input RGB image into Basel Face Model params to reconstruct 3D face.

In order to learn the parameters of the network, we build a loss function to measure two parts of difference between generated face and original face including per-pixel color and 2D landmarks.

To measure the per-pixel color difference, we need to render 3D object to 2D image, then compute the per-pixel color difference. The second part is difference of 2D landmarks which represents the feature of face shape.

A. ResNet34

Residual neural networks can skip connections, or shortcut to jump over some layers. This make ResNet can avoid the problem of vanishing gradients, by reusing activation from a previous layer until the adjacent layer learns its weights.

In this paper, we are using ResNet-34, it’s a deep convolutional neural network with 34 layers. The input of our network is a 224224 RGB human face image. The output is a 1230 parameters that contains params of BFM, 3D object’s rotation set, and the lighting.

B. per-pixel color Loss

Since we can’t directly compute the difference between original true face images and reconstructed 3D face objects, we need to project the 3D face object onto a plane to get image with size 224*224 which is the same with the original face picture.

We use a virtual camera called soft-renderer (reference), which can generate a projection of the 3D object while keep its gradient traceable.

Then we use mean square error (MSE) to calculate the squared per-pixel color difference between predicted face and original face.

Here are the results of projected faces.

C. Landmark Loss

The second part is the landmark differences between original face and reconstructed face.

Facial landmarks are used to localize and represent salient regions of the face, which can help us make sure the generated face have same facial structures with the original face.

We use face-alignment to detect 68 landmarks of the training images. During training, we project the 3D landmarks vertices of our reconstructed shape onto the image to obtain predicted 2D landmarks.

We use Smooth mean absolute error function to measure the difference between two facial landmarks.

Here are the results of facial landmarks. The first image is the facial landmarks of projected face. The second image is the facial landmarks of original face.

D. Regularization

To prevent face shape and texture degeneration, we add a commonly used loss on the regressed 3DMM coefficients: which enforces a prior distribution towards the mean face. The balancing weights are empirically set to α=10, β=0.5.

E. Training

We use 50000 images to train our network for 0 epochs. The loss of the net decreased from 22 to 0.41.

V. Unity part

In this part, we will run our trained 3D face reconstruction network in Unity. Because Unity’s programming language is C#, we can’t run python program directly. To access to python file, we need to use C#’s class: Diagnostics.

The class provides access to local and remote processes and enables us to start and stop local system processes. We can use the diagnostics class to open the Windows command prompt and execute commands, create a process and then invokes the command that we want to execute.

Then we can encapsulate codes that can execute python file in Unity, which means we can use out trained 3D face reconstruction network through unity.
