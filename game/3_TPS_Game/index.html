使用魔法的第三人称射击游戏Demo

看了一部讲述科技文明与魔法文明对抗的小说后，觉得科技对抗魔法的射击游戏会很有意思，于是尝试着制作了这款第三人称魔法射击游戏的Demo。

Demo使用UE4制作，代码部分实现通过C++和蓝图。

  1. 魔法的技能特效部分使用Variety FX Pack，其余部分使用Niagara自制。
  2. 敌人AI使用Behavior Tree和EQS(Environment Query System)实现。
  3. 立体机动部分通过cable component实现。

Version 0.1

schedule

2021-12-15

完成第三人称的枪械射击和敌人AI部分。

主角使用来复枪，通过R键更换子弹，在五秒未受到伤害后会开启自动回血，屏幕中间的血量提示UI会在玩家血量低于50%后出现。

敌人和玩家使用相同枪械，射击后会首先从角色摄像头位置发出线性碰撞检测，然后在终点位置和枪口间生成子弹轨迹特效，通过Physics Material，如果命中敌人会在命中位置生成血液特效，如果命中头部会有爆头效果。

敌人士兵AI使用行为树和EQS(Environment Query System)实现，敌人会在玩家和自己之间寻找掩体，每次射击以后都会通过EQS寻找一个可移动点，移动后再次射击。

自爆球敌人会朝着玩家方向移动，一旦靠近玩家会自爆并且造成伤害。

击杀敌人会产生分数。积分在左上角。

2020-12-16

将玩家的攻击从枪械射击更换为火球术，按下鼠标左键攻击，然后会在玩家角色前100距离的位置生成火球，火球在LifeSpan耗尽前，如果发生碰撞会发生爆炸，造成范围内的爆炸伤害。

2020-12-18

使用Niagara系统自制激光粒子特效。

实现了玩家的激光攻击，按下右键后，会从玩家右手的Socket位置朝着摄像头方向发出激光，在命中位置生成粒子特效并造成伤害。

2020-12-20

使用虚幻商城的Variety FX Pack和Niagara系统完成陨石和爆炸特效。

实现了玩家的陨石魔法攻击，按下q键后，可以拖动鼠标，选择陨石下落位置，再次按下q键后释放陨石魔法，下落后在命中位置爆炸并造成范围伤害。

2020-12-22

在看完进击的巨人最新季后，决定在游戏内实现立体机动效果。

立体机动装置使用UE4中的cable component实现。实现方法：

  1. 按下E键后，会沿着摄像机方向发出碰撞检测线，如果发生碰撞，则会记录Target Position。
  2. 命中后，通过更新cable的End Position,并延长绳索长度，让绳索朝着目标位置发射过去。
  3. 绳索命中后，通过Lanuch Character函数让角色沿着绳索方向位移。在此期间需要更新绳索的长度。
  4. 玩家角色靠近Target后，将cable恢复为默认状态，立体机动完成。

2020-2-21

增加了法力值系统，现在玩家释放的所有魔法都需要消耗法力值，法力值会随着时间自动恢复。

添加了近战的敌人。
